# 📖 Sistema de Gerenciamento de Cursos

## 📖 Sobre o Projeto
O Sistema de Gerenciamento de Cursos foi criado com o objetivo de facilitar o controle e a administração de cursos, alunos e informações relacionadas ao ambiente educacional. A plataforma oferece uma interface moderna, organizada e responsiva, proporcionando uma experiência agradável para os usuários.

Sistemas desse tipo são amplamente utilizados para centralizar informações acadêmicas e otimizar processos administrativos.

---

## 🚀 Funcionalidades
- Cadastro de cursos  
- Listagem de cursos cadastrados  
- Visualização das informações dos cursos  
- Interface intuitiva e responsiva  
- Navegação simples e rápida  
- Organização centralizada dos dados  

---

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Git  
- GitHub Pages  

O projeto está hospedado utilizando o GitHub Pages, plataforma que permite publicar sites estáticos diretamente a partir de um repositório GitHub.
